Política de Privacidade

Última atualização: Outubro 2025

Versão: 1.0

1. Introdução

Jynx ("nós", "nosso", "o aplicativo") é uma plataforma de gaming social que permite aos jogadores se conectarem, formarem equipes e jogarem juntos.

Informações Legais:

  • Entidade Legal: Jynx Group Jynx Group
  • Contato do Oficial de Privacidade: dpo@jynx.app dpo@jynx.app
  • Jurisdição: França, União Europeia (Conformidade GDPR) France, European Union (GDPR Compliance)

Esta política de privacidade descreve como coletamos, usamos, compartilhamos e protegemos seus dados pessoais quando você usa nosso aplicativo.

2. Idade Mínima e Verificação

Nosso aplicativo é reservado para usuários com 18 anos ou mais.

Ao criar sua conta, pedimos que você forneça sua data de nascimento. Se você tiver menos de 18 anos, não poderá criar uma conta ou usar nossos serviços.

3. Dados Coletados

3.1. Dados de Registro e Perfil

Coletamos as seguintes informações ao criar e gerenciar sua conta:

Informações de Autenticação:

  • Email (para contas email/senha)
  • Nome de usuário/apelido
  • Foto de perfil (opcional)
  • Identificadores de login social (Google, Apple) se você usar esses métodos

Informações do Perfil:

  • Data de nascimento (para verificação de idade)
  • País e fuso horário
  • Idiomas falados
  • Biografia (opcional)
  • Preferências de gaming e disponibilidade
  • Identificador único (gerado automaticamente para descoberta privada de amigos)

3.2. Dados de Gaming

Para ajudá-lo a encontrar companheiros compatíveis, coletamos:

Perfis de Jogo:

  • Jogos que você joga
  • Níveis, ranks e estatísticas de jogo
  • Campeões/personagens favoritos
  • Estilo de jogo

Integração com Riot Games API:

Recuperamos da Riot Games API (com seu consentimento):

  • Nome de invocador e Riot ID
  • Rank e histórico de partidas (League of Legends, Valorant, etc.)
  • Estatísticas de campeões
  • Histórico de partidas recente

Razão da Coleta: Exibir seu perfil de jogador e melhorar a correspondência com outros jogadores. Display your player profile and improve matching with other players.

3.3. Dados de Uso e Técnicos

Coletamos automaticamente certos dados técnicos:

Analytics e Desempenho:

  • Firebase Performance Monitoring: Tempo de inicialização do app, fluidez da interface App startup time, interface fluidity
  • Sentry: Relatórios de erros e desempenho (com anonimização de dados pessoais) Error and performance reports (with personal data anonymization)
  • Firebase Crashlytics: Relatórios de falhas Crash reports

Informações do Dispositivo:

  • Tipo e modelo de dispositivo
  • Versão do sistema operacional
  • Versão do aplicativo

Comportamento do Usuário:

  • Telas visitadas e recursos usados
  • Tempo gasto no aplicativo
  • Interações com a interface

Você pode desabilitar a coleta de dados de analytics em Configurações > Privacidade > "Habilitar Analytics".

3.4. Dados de Comunicação

  • Mensagens: Mensagens em chats de comunidade, mensagens privadas entre usuários, histórico de conversação Messages in community chats, private messages between users, conversation history
  • Notificações: Tokens FCM (Firebase Cloud Messaging) para notificações push FCM tokens (Firebase Cloud Messaging) for push notifications

3.5. Conteúdo Gerado pelo Usuário

  • Fotos enviadas (imagens de perfil, fotos adicionais)
  • Relatórios de segurança e moderação
  • Preferências cosméticas (avatares, emblemas)

4. Serviços de Terceiros e Compartilhamento de Dados

4.1. Serviços Firebase (Google)

Usamos os seguintes serviços Firebase, todos em conformidade com GDPR:

  • Firebase Authentication: Gerenciamento seguro de contas de usuário Secure user account management
  • Cloud Firestore: Armazenamento de seu perfil e dados de jogo Storage of your profile and game data
  • Firebase Storage: Hospedagem de suas imagens de perfil Hosting your profile images
  • Cloud Functions: Execução segura de lógica backend Secure backend logic execution
  • Firebase Cloud Messaging: Notificações push Push notifications
  • Firebase App Check: Proteção contra abuso e bots Protection against abuse and bots
  • Firebase Performance Monitoring: Monitoramento de desempenho do app App performance monitoring
  • Firebase Crashlytics: Relatórios de falhas Crash reports

Política de Privacidade Firebase: https://firebase.google.com/support/privacy https://firebase.google.com/support/privacy

4.2. Sentry (Monitoramento de Erros)

Usamos Sentry para rastreamento de erros e desempenho. Os dados coletados incluem:

  • Rastreamentos de erros
  • Contexto técnico (SO, versão do app)

Política de Privacidade Sentry: https://sentry.io/privacy/ https://sentry.io/privacy/

4.3. Riot Games API

Quando você vincula sua conta Riot Games, recuperamos suas estatísticas de jogo via API oficial da Riot Games. Estes dados são usados apenas para exibir seu perfil e melhorar a correspondência.

Política de Privacidade Riot Games: https://www.riotgames.com/en/privacy-notice https://www.riotgames.com/en/privacy-notice

5. Uso de Dados

Usamos seus dados para:

1. Fornecer Nossos Serviços:

  • Criar e gerenciar sua conta
  • Permitir que você se conecte com outros jogadores
  • Facilitar criação de equipes e organização de sessões de jogo

2. Personalização:

  • Recomendar companheiros compatíveis (correspondência baseada em jogo, rank, estilo)
  • Exibir conteúdo relevante

3. Comunicação:

  • Enviar notificações importantes (correspondências encontradas, mensagens recebidas)

4. Melhoria do Serviço:

  • Analisar uso para melhorar recursos
  • Identificar e corrigir bugs
  • Otimizar desempenho

5. Segurança e Conformidade:

  • Prevenir fraude e abuso
  • Fazer cumprir nossos termos de uso
  • Cumprir obrigações legais

6. Retenção de Dados

6.1. Períodos de Retenção

  • Dados de Perfil: Retidos enquanto sua conta estiver ativa Retained as long as your account is active
  • Registros e Analytics: Firebase Analytics: 14 meses (política Firebase) Firebase Analytics: 14 months (Firebase policy)

6.2. Exclusão de Dados

Você pode solicitar a exclusão de sua conta a qualquer momento (veja Seção 7).

Retenção Legal: Alguns dados podem ser retidos por mais tempo para cumprir nossas obrigações legais (ex: relatórios de segurança, transações financeiras). Some data may be retained longer to comply with our legal obligations (e.g., security reports, financial transactions).

7. Seus Direitos (GDPR)

Sob o GDPR, você tem os seguintes direitos:

7.1. Direito de Acesso

Você pode solicitar uma cópia de todos os dados que temos sobre você.

Como exercer este direito: Configurações > Privacidade > "Exportar Meus Dados" Settings > Privacy > "Export My Data"

O processo gera um arquivo JSON contendo todos os seus dados (perfil, mensagens, estatísticas, etc.). O link de download é válido por 7 dias.

7.2. Direito de Retificação

Você pode modificar as informações do seu perfil a qualquer momento pelo aplicativo (nome de usuário, biografia, preferências, etc.).

7.3. Direito ao Esquecimento ("Direito de Ser Esquecido")

Você pode solicitar a exclusão de sua conta.

Como exercer este direito: Configurações > Zona de Perigo > "Excluir Minha Conta" Settings > Danger Zone > "Delete My Account"

Processo de Exclusão Automatizado:

  1. Confirmação em duas etapas para prevenir exclusões acidentais
  2. Anonimização imediata de seus dados pessoais (Firestore)
  3. Exclusão de sua conta Firebase Authentication (revoga automaticamente todos os tokens OAuth)
  4. Um ID anonimizado é fornecido como referência
  5. Opção para excluir suas mensagens e conteúdo gerado

Importante: A exclusão é irreversível. Alguns dados anonimizados podem ser retidos para fins estatísticos (em conformidade com GDPR). Deletion is irreversible. Some anonymized data may be retained for statistical purposes (GDPR-compliant).

Solicitações de Exclusão Manual: Para solicitações de exclusão de dados específicos (ex: excluir apenas mensagens de uma comunidade específica, excluir imagens do Storage, ou outros casos especiais), entre em contato com nosso Oficial de Proteção de Dados em dpo@jynx.app. Processaremos solicitações manuais dentro de 30 dias conforme exigido pelo GDPR. For specific data deletion requests (e.g., delete only messages from a specific community, delete Storage images, or other edge cases), please contact our Data Protection Officer at dpo@jynx.app. We will process manual requests within 30 days as required by GDPR.

7.4. Direito à Portabilidade de Dados

A exportação de dados (Seção 7.1) permite que você recupere seus dados em formato estruturado e legível por máquina (JSON).

7.5. Direito de Objeção

Você pode se opor ao processamento de seus dados para fins de marketing direto.

7.6. Direito de Retirar Consentimento

Você pode retirar seu consentimento para:

  • Analytics: Configurações > Privacidade > Desabilitar "Habilitar Analytics" Settings > Privacy > Disable "Enable Analytics"
  • Notificações Push: Configurações do dispositivo Device settings

Nota: Tokens OAuth (Google, Apple) são automaticamente revogados quando você exclui sua conta (Seção 7.3). OAuth tokens (Google, Apple) are automatically revoked when you delete your account (Section 7.3).

8. Segurança de Dados

Implementamos medidas de segurança técnicas e organizacionais para proteger seus dados:

  • Criptografia: Comunicação segura via HTTPS Secure communication via HTTPS
  • Regras de Segurança Firebase: Controle de acesso rigoroso aos dados do Firestore Strict access control to Firestore data
  • Autenticação: Gerenciamento seguro de sessão via Firebase Auth Secure session management via Firebase Auth
  • Validação de Entrada: Prevenção de injeção XSS via SecureTextInput XSS injection prevention via SecureTextInput
  • Limitação de Taxa: Proteção contra abuso (mensagens spam, ações automatizadas) Protection against abuse (spam messages, automated actions)
  • Moderação: Sistema de relatório de conteúdo e moderação Content reporting and moderation system

Nenhum sistema é 100% seguro. Em caso de violação de dados, notificaremos você em conformidade com GDPR (dentro de 72 horas). In case of a data breach, we will notify you in compliance with GDPR (within 72 hours).

9. Transferências Internacionais de Dados

Seus dados podem ser transferidos e armazenados em servidores localizados fora da União Europeia, incluindo:

  • Google Cloud / Firebase: Usa infraestrutura global com conformidade GDPR Uses global infrastructure with GDPR compliance
  • Sentry: Servidores baseados nos EUA com conformidade Privacy Shield US-based servers with Privacy Shield compliance

Garantias: Todos os nossos parceiros oferecem garantias de proteção equivalentes ao GDPR (cláusulas contratuais padrão, certificações Privacy Shield). All our partners offer protection guarantees equivalent to GDPR (standard contractual clauses, Privacy Shield certifications).

10. Cookies e Tecnologias Similares

Nosso aplicativo móvel não usa cookies no sentido tradicional, mas usa:

  • Tokens de Autenticação: Armazenados localmente de forma segura Stored locally in a secure manner
  • Cache Local: Para melhorar desempenho (dados de perfil, mensagens recentes) To improve performance (profile data, recent messages)

11. Mudanças nesta Política

Podemos atualizar esta política de privacidade. Em caso de mudanças importantes, notificaremos você:

  • Via notificação no aplicativo
  • Por email no endereço associado à sua conta
