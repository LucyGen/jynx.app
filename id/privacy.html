Kebijakan Privasi

Terakhir diperbarui: Oktober 2025

Versi: 1.0

1. Pendahuluan

Jynx ("kami", "kami", "aplikasi") adalah platform gaming sosial yang memungkinkan pemain untuk terhubung, membentuk tim, dan bermain bersama.

Informasi Hukum:

  • Entitas Hukum: Jynx Group Jynx Group
  • Kontak Petugas Privasi: dpo@jynx.app dpo@jynx.app
  • Yurisdiksi: Prancis, Uni Eropa (Kepatuhan GDPR) France, European Union (GDPR Compliance)

Kebijakan privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, berbagi, dan melindungi data pribadi Anda saat Anda menggunakan aplikasi kami.

2. Usia Minimum dan Verifikasi

Aplikasi kami diperuntukkan bagi pengguna berusia 18 tahun ke atas.

Saat membuat akun Anda, kami meminta Anda untuk memberikan tanggal lahir. Jika Anda berusia di bawah 18 tahun, Anda tidak akan dapat membuat akun atau menggunakan layanan kami.

3. Data yang Dikumpulkan

3.1. Data Registrasi dan Profil

Kami mengumpulkan informasi berikut saat membuat dan mengelola akun Anda:

Informasi Autentikasi:

  • Email (untuk akun email/password)
  • Username/nickname
  • Foto profil (opsional)
  • Identifier login sosial (Google, Apple) jika Anda menggunakan metode ini

Informasi Profil:

  • Tanggal lahir (untuk verifikasi usia)
  • Negara dan zona waktu
  • Bahasa yang digunakan
  • Biografi (opsional)
  • Preferensi gaming dan ketersediaan
  • Handle unik (dibuat otomatis untuk penemuan teman pribadi)

3.2. Data Gaming

Untuk membantu Anda menemukan rekan tim yang kompatibel, kami mengumpulkan:

Profil Game:

  • Game yang Anda mainkan
  • Level, rank, dan statistik game
  • Champion/karakter favorit
  • Gaya bermain

Integrasi Riot Games API:

Kami mengambil dari Riot Games API (dengan persetujuan Anda):

  • Nama summoner dan Riot ID
  • Rank dan riwayat pertandingan (League of Legends, Valorant, dll.)
  • Statistik champion
  • Riwayat pertandingan terkini

Alasan Pengumpulan: Menampilkan profil pemain Anda dan meningkatkan matching dengan pemain lain. Display your player profile and improve matching with other players.

3.3. Data Penggunaan dan Teknis

Kami secara otomatis mengumpulkan data teknis tertentu:

Analitik dan Performa:

  • Firebase Performance Monitoring: Waktu startup aplikasi, kelancaran antarmuka App startup time, interface fluidity
  • Sentry: Laporan error dan performa (dengan anonimisasi data pribadi) Error and performance reports (with personal data anonymization)
  • Firebase Crashlytics: Laporan crash Crash reports

Informasi Perangkat:

  • Jenis dan model perangkat
  • Versi sistem operasi
  • Versi aplikasi

Perilaku Pengguna:

  • Layar yang dikunjungi dan fitur yang digunakan
  • Waktu yang dihabiskan di aplikasi
  • Interaksi antarmuka

Anda dapat menonaktifkan pengumpulan data analitik di Pengaturan > Privasi > "Aktifkan Analitik".

3.4. Data Komunikasi

  • Pesan: Pesan dalam chat komunitas, pesan pribadi antar pengguna, riwayat percakapan Messages in community chats, private messages between users, conversation history
  • Notifikasi: Token FCM (Firebase Cloud Messaging) untuk notifikasi push FCM tokens (Firebase Cloud Messaging) for push notifications

3.5. Konten yang Dibuat Pengguna

  • Foto yang diunggah (gambar profil, foto tambahan)
  • Laporan keamanan dan moderasi
  • Preferensi kosmetik (avatar, badge)

4. Layanan Pihak Ketiga dan Berbagi Data

4.1. Layanan Firebase (Google)

Kami menggunakan layanan Firebase berikut, semua mematuhi GDPR:

  • Firebase Authentication: Manajemen akun pengguna yang aman Secure user account management
  • Cloud Firestore: Penyimpanan profil dan data game Anda Storage of your profile and game data
  • Firebase Storage: Hosting gambar profil Anda Hosting your profile images
  • Cloud Functions: Eksekusi logika backend yang aman Secure backend logic execution
  • Firebase Cloud Messaging: Notifikasi push Push notifications
  • Firebase App Check: Perlindungan terhadap penyalahgunaan dan bot Protection against abuse and bots
  • Firebase Performance Monitoring: Pemantauan performa aplikasi App performance monitoring
  • Firebase Crashlytics: Laporan crash Crash reports

Kebijakan Privasi Firebase: https://firebase.google.com/support/privacy https://firebase.google.com/support/privacy

4.2. Sentry (Pemantauan Error)

Kami menggunakan Sentry untuk pelacakan error dan performa. Data yang dikumpulkan termasuk:

  • Jejak error
  • Konteks teknis (OS, versi aplikasi)

Kebijakan Privasi Sentry: https://sentry.io/privacy/ https://sentry.io/privacy/

4.3. Riot Games API

Saat Anda menghubungkan akun Riot Games, kami mengambil statistik game Anda melalui API resmi Riot Games. Data ini hanya digunakan untuk menampilkan profil Anda dan meningkatkan matching.

Kebijakan Privasi Riot Games: https://www.riotgames.com/en/privacy-notice https://www.riotgames.com/en/privacy-notice

5. Penggunaan Data

Kami menggunakan data Anda untuk:

1. Menyediakan Layanan Kami:

  • Membuat dan mengelola akun Anda
  • Memungkinkan Anda terhubung dengan pemain lain
  • Memfasilitasi pembuatan tim dan organisasi sesi gaming

2. Personalisasi:

  • Merekomendasikan rekan tim yang kompatibel (matching berbasis game, rank, gaya)
  • Menampilkan konten yang relevan

3. Komunikasi:

  • Mengirim notifikasi penting (match ditemukan, pesan diterima)

4. Peningkatan Layanan:

  • Menganalisis penggunaan untuk meningkatkan fitur
  • Mengidentifikasi dan memperbaiki bug
  • Mengoptimalkan performa

5. Keamanan dan Kepatuhan:

  • Mencegah penipuan dan penyalahgunaan
  • Menegakkan ketentuan penggunaan kami
  • Mematuhi kewajiban hukum

6. Penyimpanan Data

6.1. Periode Penyimpanan

  • Data Profil: Disimpan selama akun Anda aktif Retained as long as your account is active
  • Log dan Analitik: Firebase Analytics: 14 bulan (kebijakan Firebase) Firebase Analytics: 14 months (Firebase policy)

6.2. Penghapusan Data

Anda dapat meminta penghapusan akun Anda kapan saja (lihat Bagian 7).

Penyimpanan Hukum: Beberapa data mungkin disimpan lebih lama untuk mematuhi kewajiban hukum kami (misalnya, laporan keamanan, transaksi keuangan). Some data may be retained longer to comply with our legal obligations (e.g., security reports, financial transactions).

7. Hak Anda (GDPR)

Berdasarkan GDPR, Anda memiliki hak-hak berikut:

7.1. Hak Akses

Anda dapat meminta salinan semua data yang kami simpan tentang Anda.

Cara menggunakan hak ini: Pengaturan > Privasi > "Ekspor Data Saya" Settings > Privacy > "Export My Data"

Proses ini menghasilkan file JSON yang berisi semua data Anda (profil, pesan, statistik, dll.). Link download berlaku selama 7 hari.

7.2. Hak Pembetulan

Anda dapat mengubah informasi profil Anda kapan saja dari aplikasi (username, bio, preferensi, dll.).

7.3. Hak Penghapusan ("Hak untuk Dilupakan")

Anda dapat meminta penghapusan akun Anda.

Cara menggunakan hak ini: Pengaturan > Zona Bahaya > "Hapus Akun Saya" Settings > Danger Zone > "Delete My Account"

Proses Penghapusan Otomatis:

  1. Konfirmasi dua langkah untuk mencegah penghapusan tidak sengaja
  2. Anonimisasi langsung data pribadi Anda (Firestore)
  3. Penghapusan akun Firebase Authentication Anda (secara otomatis mencabut semua token OAuth)
  4. ID anonim disediakan sebagai referensi
  5. Opsi untuk menghapus pesan dan konten yang dibuat

Penting: Penghapusan tidak dapat dibatalkan. Beberapa data anonim mungkin disimpan untuk tujuan statistik (mematuhi GDPR). Deletion is irreversible. Some anonymized data may be retained for statistical purposes (GDPR-compliant).

Permintaan Penghapusan Manual: Untuk permintaan penghapusan data khusus (misalnya, hapus hanya pesan dari komunitas tertentu, hapus gambar Storage, atau kasus lainnya), silakan hubungi Petugas Perlindungan Data kami di dpo@jynx.app. Kami akan memproses permintaan manual dalam 30 hari sesuai dengan GDPR. For specific data deletion requests (e.g., delete only messages from a specific community, delete Storage images, or other edge cases), please contact our Data Protection Officer at dpo@jynx.app. We will process manual requests within 30 days as required by GDPR.

7.4. Hak Portabilitas Data

Ekspor data (Bagian 7.1) memungkinkan Anda mengambil data dalam format terstruktur yang dapat dibaca mesin (JSON).

7.5. Hak untuk Menolak

Anda dapat menolak pemrosesan data Anda untuk tujuan pemasaran langsung.

7.6. Hak untuk Menarik Persetujuan

Anda dapat menarik persetujuan untuk:

  • Analitik: Pengaturan > Privasi > Nonaktifkan "Aktifkan Analitik" Settings > Privacy > Disable "Enable Analytics"
  • Notifikasi Push: Pengaturan perangkat Device settings

Catatan: Token OAuth (Google, Apple) secara otomatis dicabut saat Anda menghapus akun (Bagian 7.3). OAuth tokens (Google, Apple) are automatically revoked when you delete your account (Section 7.3).

8. Keamanan Data

Kami menerapkan langkah-langkah keamanan teknis dan organisasi untuk melindungi data Anda:

  • Enkripsi: Komunikasi aman melalui HTTPS Secure communication via HTTPS
  • Firebase Security Rules: Kontrol akses ketat ke data Firestore Strict access control to Firestore data
  • Autentikasi: Manajemen sesi aman melalui Firebase Auth Secure session management via Firebase Auth
  • Validasi Input: Pencegahan injeksi XSS melalui SecureTextInput XSS injection prevention via SecureTextInput
  • Rate Limiting: Perlindungan terhadap penyalahgunaan (spam pesan, tindakan otomatis) Protection against abuse (spam messages, automated actions)
  • Moderasi: Sistem pelaporan dan moderasi konten Content reporting and moderation system

Tidak ada sistem yang 100% aman. Jika terjadi pelanggaran data, kami akan memberi tahu Anda sesuai dengan GDPR (dalam 72 jam). In case of a data breach, we will notify you in compliance with GDPR (within 72 hours).

9. Transfer Data Internasional

Data Anda mungkin ditransfer dan disimpan di server yang berlokasi di luar Uni Eropa, termasuk:

  • Google Cloud / Firebase: Menggunakan infrastruktur global dengan kepatuhan GDPR Uses global infrastructure with GDPR compliance
  • Sentry: Server berbasis AS dengan kepatuhan Privacy Shield US-based servers with Privacy Shield compliance

Jaminan: Semua mitra kami menawarkan jaminan perlindungan setara dengan GDPR (klausul kontrak standar, sertifikasi Privacy Shield). All our partners offer protection guarantees equivalent to GDPR (standard contractual clauses, Privacy Shield certifications).

10. Cookie dan Teknologi Serupa

Aplikasi mobile kami tidak menggunakan cookie dalam arti tradisional, tetapi menggunakan:

  • Token Autentikasi: Disimpan secara lokal dengan cara yang aman Stored locally in a secure manner
  • Cache Lokal: Untuk meningkatkan performa (data profil, pesan terkini) To improve performance (profile data, recent messages)

11. Perubahan pada Kebijakan Ini

Kami dapat memperbarui kebijakan privasi ini. Jika ada perubahan besar, kami akan memberi tahu Anda:

  • Melalui notifikasi dalam aplikasi
  • Melalui email ke alamat yang terkait dengan akun Anda

12. Hubungi Kami

Untuk pertanyaan mengenai kebijakan privasi ini atau untuk menggunakan hak Anda:

  • Email: dpo@jynx.app dpo@jynx.app

Otoritas Pengawas: Jika Anda percaya hak Anda tidak dihormati, Anda dapat mengajukan keluhan ke CNIL (Commission Nationale de l'Informatique et des Libertés) di Prancis. If you believe your rights are not being respected, you can file a complaint with the CNIL (Commission Nationale de l'Informatique et des Libertés) in France.

13. Persetujuan

Dengan menggunakan Jynx, Anda menyetujui pengumpulan dan penggunaan data Anda sebagaimana dijelaskan dalam kebijakan privasi ini.

Anda dapat menarik persetujuan kapan saja dengan menghapus akun Anda.
