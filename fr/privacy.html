Politique de Confidentialité

Dernière mise à jour : Octobre 2025

Version : 1.0

1. Introduction

Jynx ("nous", "notre", "l'application") est une plateforme sociale de gaming qui permet aux joueurs de se connecter, de former des équipes et de jouer ensemble.

Informations Légales :

  • Entité Légale : Jynx Group Jynx Group
  • Contact Responsable de la Protection des Données : dpo@jynx.app dpo@jynx.app
  • Juridiction : France, Union Européenne (Conformité RGPD) France, European Union (GDPR Compliance)

Cette politique de confidentialité décrit comment nous collectons, utilisons, partageons et protégeons vos données personnelles lorsque vous utilisez notre application.

2. Âge Minimum et Vérification

Notre application est réservée aux utilisateurs âgés de 18 ans et plus.

Lors de la création de votre compte, nous vous demandons de fournir votre date de naissance. Si vous avez moins de 18 ans, vous ne pourrez pas créer de compte ou utiliser nos services.

3. Données Collectées

3.1. Données d'Inscription et de Profil

Nous collectons les informations suivantes lors de la création et de la gestion de votre compte :

Informations d'Authentification :

  • Email (pour les comptes email/mot de passe)
  • Nom d'utilisateur/pseudo
  • Photo de profil (optionnelle)
  • Identifiants de connexion sociale (Google, Apple) si vous utilisez ces méthodes

Informations de Profil :

  • Date de naissance (pour vérification d'âge)
  • Pays et fuseau horaire
  • Langues parlées
  • Biographie (optionnelle)
  • Préférences de jeu et disponibilités
  • Identifiant unique (généré automatiquement pour découverte privée d'amis)

3.2. Données de Jeu

Pour vous aider à trouver des coéquipiers compatibles, nous collectons :

Profils de Jeu :

  • Jeux auxquels vous jouez
  • Niveaux, rangs et statistiques de jeu
  • Champions/personnages favoris
  • Style de jeu

Intégration API Riot Games :

Nous récupérons depuis l'API Riot Games (avec votre consentement) :

  • Nom d'invocateur et Riot ID
  • Rang et historique de matchs (League of Legends, Valorant, etc.)
  • Statistiques de champions
  • Historique de matchs récents

Raison de la Collecte : Afficher votre profil de joueur et améliorer le matching avec d'autres joueurs. Display your player profile and improve matching with other players.

3.3. Données d'Utilisation et Techniques

Nous collectons automatiquement certaines données techniques :

Analytiques et Performance :

  • Firebase Performance Monitoring : Temps de démarrage de l'app, fluidité de l'interface App startup time, interface fluidity
  • Sentry : Rapports d'erreurs et de performance (avec anonymisation des données personnelles) Error and performance reports (with personal data anonymization)
  • Firebase Crashlytics : Rapports de crash Crash reports

Informations de l'Appareil :

  • Type et modèle d'appareil
  • Version du système d'exploitation
  • Version de l'application

Comportement Utilisateur :

  • Écrans visités et fonctionnalités utilisées
  • Temps passé dans l'application
  • Interactions avec l'interface

Vous pouvez désactiver la collecte de données analytiques dans Paramètres > Confidentialité > "Activer les Analytiques".

3.4. Données de Communication

  • Messages : Messages dans les chats communautaires, messages privés entre utilisateurs, historique de conversations Messages in community chats, private messages between users, conversation history
  • Notifications : Jetons FCM (Firebase Cloud Messaging) pour notifications push FCM tokens (Firebase Cloud Messaging) for push notifications

3.5. Contenu Généré par l'Utilisateur

  • Photos téléchargées (photos de profil, photos supplémentaires)
  • Rapports de sécurité et modération
  • Préférences cosmétiques (avatars, badges)

4. Services Tiers et Partage de Données

4.1. Services Firebase (Google)

Nous utilisons les services Firebase suivants, tous conformes au RGPD :

  • Firebase Authentication : Gestion sécurisée des comptes utilisateurs Secure user account management
  • Cloud Firestore : Stockage de votre profil et données de jeu Storage of your profile and game data
  • Firebase Storage : Hébergement de vos images de profil Hosting your profile images
  • Cloud Functions : Exécution sécurisée de la logique backend Secure backend logic execution
  • Firebase Cloud Messaging : Notifications push Push notifications
  • Firebase App Check : Protection contre les abus et bots Protection against abuse and bots
  • Firebase Performance Monitoring : Surveillance des performances de l'app App performance monitoring
  • Firebase Crashlytics : Rapports de crash Crash reports

Politique de Confidentialité Firebase : https://firebase.google.com/support/privacy https://firebase.google.com/support/privacy

4.2. Sentry (Surveillance des Erreurs)

Nous utilisons Sentry pour le suivi des erreurs et performances. Les données collectées incluent :

  • Traces d'erreurs
  • Contexte technique (OS, version de l'app)

Politique de Confidentialité Sentry : https://sentry.io/privacy/ https://sentry.io/privacy/

4.3. API Riot Games

Lorsque vous liez votre compte Riot Games, nous récupérons vos statistiques de jeu via l'API officielle Riot Games. Ces données sont utilisées uniquement pour afficher votre profil et améliorer le matching.

Politique de Confidentialité Riot Games : https://www.riotgames.com/en/privacy-notice https://www.riotgames.com/en/privacy-notice

Nous utilisons vos données pour :

1. Fournir Nos Services :

  • Créer et gérer votre compte
  • Vous permettre de vous connecter avec d'autres joueurs
  • Faciliter la création d'équipes et l'organisation de sessions de jeu

2. Personnalisation :

  • Recommander des coéquipiers compatibles (matching basé sur le jeu, le rang, le style)
  • Afficher du contenu pertinent

3. Communication :

  • Vous envoyer des notifications importantes (matchs trouvés, messages reçus)

4. Amélioration du Service :

  • Analyser l'utilisation pour améliorer les fonctionnalités

  • Optimiser les performances

5. Sécurité et Conformité :

  • Prévenir la fraude et les abus
  • Appliquer nos conditions d'utilisation
  • Respecter nos obligations légales

6. Conservation des Données

6.1. Périodes de Conservation

  • Données de Profil : Conservées tant que votre compte est actif Retained as long as your account is active
  • Logs et Analytiques : Firebase Analytics : 14 mois (politique Firebase) Firebase Analytics: 14 months (Firebase policy)

6.2. Suppression des Données

Vous pouvez demander la suppression de votre compte à tout moment (voir Section 7).

Conservation Légale : Certaines données peuvent être conservées plus longtemps pour respecter nos obligations légales (par ex., rapports de sécurité, transactions financières). Some data may be retained longer to comply with our legal obligations (e.g., security reports, financial transactions).

7. Vos Droits (RGPD)

Selon le RGPD, vous disposez des droits suivants :

7.1. Droit d'Accès

Vous pouvez demander une copie de toutes les données que nous détenons sur vous.

Comment exercer ce droit : Paramètres > Confidentialité > "Exporter Mes Données" Settings > Privacy > "Export My Data"

Le processus génère un fichier JSON contenant toutes vos données (profil, messages, statistiques, etc.). Le lien de téléchargement est valide 7 jours.

7.2. Droit de Rectification

Vous pouvez modifier vos informations de profil à tout moment depuis l'application (nom d'utilisateur, bio, préférences, etc.).

7.3. Droit à l'Effacement ("Droit à l'Oubli")

Vous pouvez demander la suppression de votre compte.

Comment exercer ce droit : Paramètres > Zone de Danger > "Supprimer Mon Compte" Settings > Danger Zone > "Delete My Account"

Processus de Suppression Automatisé :

  1. Confirmation en deux étapes pour éviter les suppressions accidentelles
  2. Anonymisation immédiate de vos données personnelles (Firestore)
  3. Suppression de votre compte Firebase Authentication (révoque automatiquement tous les jetons OAuth)
  4. Un ID anonymisé est fourni comme référence
  5. Option de supprimer vos messages et contenu généré

Important : La suppression est irréversible. Certaines données anonymisées peuvent être conservées à des fins statistiques (conforme RGPD). Deletion is irreversible. Some anonymized data may be retained for statistical purposes (GDPR-compliant).

Demandes de Suppression Manuelle : Pour des demandes de suppression de données spécifiques (par ex., supprimer uniquement les messages d'une communauté spécifique, supprimer les images Storage, ou autres cas particuliers), veuillez contacter notre Délégué à la Protection des Données à dpo@jynx.app. Nous traiterons les demandes manuelles sous 30 jours comme requis par le RGPD. For specific data deletion requests (e.g., delete only messages from a specific community, delete Storage images, or other edge cases), please contact our Data Protection Officer at dpo@jynx.app. We will process manual requests within 30 days as required by GDPR.

7.4. Droit à la Portabilité des Données

L'export de données (Section 7.1) vous permet de récupérer vos données dans un format structuré et lisible par machine (JSON).

7.5. Droit d'Opposition

Vous pouvez vous opposer au traitement de vos données à des fins de marketing direct.

7.6. Droit de Retirer le Consentement

Vous pouvez retirer votre consentement pour :

  • Analytiques : Paramètres > Confidentialité > Désactiver "Activer les Analytiques" Settings > Privacy > Disable "Enable Analytics"
  • Notifications Push : Paramètres de l'appareil Device settings

Note : Les jetons OAuth (Google, Apple) sont automatiquement révoqués lorsque vous supprimez votre compte (Section 7.3). OAuth tokens (Google, Apple) are automatically revoked when you delete your account (Section 7.3).

8. Sécurité des Données

Nous mettons en œuvre des mesures de sécurité techniques et organisationnelles pour protéger vos données :

  • Chiffrement : Communication sécurisée via HTTPS Secure communication via HTTPS
  • Règles de Sécurité Firebase : Contrôle d'accès strict aux données Firestore Strict access control to Firestore data
  • Authentification : Gestion sécurisée des sessions via Firebase Auth Secure session management via Firebase Auth
  • Validation des Entrées : Prévention des injections XSS via SecureTextInput XSS injection prevention via SecureTextInput
  • Limitation de Débit : Protection contre les abus (messages spam, actions automatisées) Protection against abuse (spam messages, automated actions)
  • Modération : Système de signalement et modération du contenu Content reporting and moderation system

Aucun système n'est sécurisé à 100%. En cas de violation de données, nous vous notifierons conformément au RGPD (sous 72 heures). In case of a data breach, we will notify you in compliance with GDPR (within 72 hours).

9. Transferts Internationaux de Données

Vos données peuvent être transférées et stockées sur des serveurs situés hors de l'Union Européenne, notamment :

  • Google Cloud / Firebase : Utilise une infrastructure mondiale avec conformité RGPD Uses global infrastructure with GDPR compliance
  • Sentry : Serveurs basés aux États-Unis avec conformité Privacy Shield US-based servers with Privacy Shield compliance

Garanties : Tous nos partenaires offrent des garanties de protection équivalentes au RGPD (clauses contractuelles types, certifications Privacy Shield). All our partners offer protection guarantees equivalent to GDPR (standard contractual clauses, Privacy Shield certifications).

10. Cookies et Technologies Similaires

Notre application mobile n'utilise pas de cookies au sens traditionnel, mais utilise :

  • Jetons d'Authentification : Stockés localement de manière sécurisée Stored locally in a secure manner
  • Cache Local : Pour améliorer les performances (données de profil, messages récents) To improve performance (profile data, recent messages)
